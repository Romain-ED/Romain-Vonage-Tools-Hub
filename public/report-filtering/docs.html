Overview

The Vonage Reports API Filter Tool is a web-based application designed to help users efficiently filter and analyze CSV reports generated by the Vonage API. This tool provides an intuitive interface for processing large CSV files with advanced filtering capabilities and data analysis features.

Purpose

This tool was created to streamline the process of working with Vonage API reports, allowing users to:

  • Filter large CSV files based on multiple criteria
  • Analyze data patterns and common values
  • Export filtered results for further processing
  • Monitor file processing with detailed logging

Key Benefits

  • No Installation Required: Runs entirely in the browser
  • Large File Support: Efficiently handles large CSV files
  • Real-time Filtering: Instant results with multiple filter criteria
  • Data Analysis: Built-in tools to understand your data
  • Export Functionality: Download filtered results as CSV

User Manual

Getting Started

  1. Upload Your CSV File:
    • Drag and drop your CSV file onto the upload area
    • Or click the upload area to browse and select a file
    • The tool accepts CSV files of any size
  2. File Processing:
    • The tool automatically parses your CSV file
    • Headers are extracted from the first row
    • Processing progress is shown in the log section

File Analysis (Optional)

  1. Select Headers to Analyze:
    • Check the boxes for columns you want to analyze
    • Analysis shows the top 10 most common values per column
  2. Run Analysis:
    • Click "Analyze Selected Headers"
    • View results with counts and percentages
    • Analysis results help with creating effective filters

Filtering Data

  1. Create Filters:
    • Select a column from the dropdown
    • Choose an operator (contains, equals, starts with, etc.)
    • Enter or select a value to filter on
    • For analyzed columns, use the suggestions dropdown
  2. Multiple Filters:
    • Click "Add Filter" to create additional filters
    • All filters are applied with AND logic
    • Remove individual filters with the "Remove" button
  3. Apply Filters:
    • Click "Apply Filters" to see results
    • Results are paginated (100 rows per page)
    • Filter statistics shown in the log

Viewing Results

  • Table View: Results displayed in a scrollable table
  • Pagination: Navigate through large result sets
  • Row Count: Total matching rows displayed
  • Export: Download filtered results as CSV

Monitoring and Logging

  • Processing Log: Real-time information about all operations
  • Log Types: Info, Success, Warning, and Error messages
  • Log Controls: Clear log or hide/show log section
  • Timestamps: Track when operations occurred

Features

  • Drag and drop or click to upload CSV files
  • Automatic CSV parsing with error handling
  • Support for files with quoted values and special characters

  • Skip malformed rows with detailed reporting

  • Sortable column headers with visual sort indicators
  • Progressive loading with background processing indicators
  • Intuitive workflow with guided steps and AI suggestions
